# Florida Liquor License Market — Next.js Development Build

A complete, tested Next.js development build based on the approved Florida Liquor License Market visual.

## Included
- Responsive header and mobile navigation
- Rebuilt hero using separate skyline and bar imagery
- Working county, license type and price filters
- Featured listing cards from typed sample data
- Action cards, statistics, transaction table, market insights and financing section
- CTA and footer
- SEO metadata

## Run locally
```bash
npm install
npm run dev
```

## Verify production build
```bash
npm run lint
npm run build
```

## Deployment safety
Upload this project to the `development` branch only. Review the Vercel preview before merging into `main`.
