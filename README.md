# Florida Liquor License Market — Responsive Homepage

This project rebuilds the approved production visual as responsive HTML, CSS, and JavaScript.

## Included
- Responsive navigation and mobile menu
- Rebuilt hero, search panel, cards, statistics, listings, market sections, CTA, and footer
- Working client-side listing filters
- Reusable image assets cropped from the approved visual

## Publish
Upload `index.html`, `styles.css`, `script.js`, and the complete `assets` folder to the root of the GitHub repository, then commit.

## Next build stage
